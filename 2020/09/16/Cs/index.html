C#入门导言

Created2020-09-16|Updated2020-11-06
|Post View:

C#

入门

基础

来源于C，C++，Java

纯面向对象语言，所有变量和方法都要在类中声明，即没有独立于类的变量和方法

环境(.Net框架

  • .NET是一个由微软所搭建的技术平台

  • .NET应用运行在.NET框架之上

  • C# 是 .Net 框架的一部分，用于编写 .Net 应用程序

CLI(Common Language Infrastructure，公共语言基层结构)：

  • 为虚拟机环境所制定的规范，为了让不同的高级语言所编制的程序可以在不同的系统环境中执行（跨平台
  • 实现方法：指定一种标准虚拟机中介语言（微软中间语言），将各种高级语言映射为该中间语言

CLR(Common Language Runtime，公共语言运行时)（类似JVM）：

  • 是CLI在PC、Windows上具体的实现，提供一个标准化环境

执行C#代码过程

  1. 选择编译器

  2. 编译为MSIL（微软中间语言）

  3. 通过实时编译器（JIT）将MSIL转为特定CPU的代码

命名空间

用于区分大型项目中很可能出现的相同的类名

例子：qz和wh各自建立了一个叫做Hello的类，这时就会产生冲突，这时有命名空间qz和wh

qz Hello和wh Hello就能解决冲突

命名空间里是一系列class

using 名词空间,可以省略在每次使用类名时的名词空间前缀

c#每一个新建工程都会自动把所用内容放到一个名字空间里，空间名就是项目名

static修饰的方法是静态函数，不用创建对象即可访问

例子：

  • Main()作为程序入口函数，如果使用之前需要创建实例对象的话不符合逻辑
  • System.Console.WriteLine()同样可以直接使用，System提供了大量的静态函数

输入输出

注意

输入的都是string，需要进一步处理成需要的数据类型

Convert.ToInt32()

控制台

Console.Write()：不自动换行

Console.WriteLine()：自动换行

Console.Read()：

Console.ReadLine()：等待直到用户按下回车，一次读入一行

Console.ReadKey()：是等待用户按下任意键，一次读入一个字符

变量

普通变量储存在栈中

引用式变量需要使用new来分配空间

例如：

  • 数组int[] num = new int[100];，num在是在栈中对托管内存的一个引用，num存一个内存索引号（num就类似指针
  • 字符串string s = "abc";在堆上创建，原型string s = new string();
  • 类

格式化输出

Console.WriteLine($"{num1} + {num2} = {result}");

var关键字

声明变量的一个占位符，在无法确定变量类型时使用

  • 必须在定义时初始化
  • 一但初始化完成，就不能再给变量赋与初始化值类型不同的值了
  • var要求是局部变量
  • 使用var定义变量在效率上和使用强类型方式定义变量完全一样

数学函数

大多是Math类的静态方法，Math类在名字空间System中

类型转换

tryParse(string s, out result)：

  • 不产生异常
  • 转换成功则返回true，转换成功的值存入result
  • 转换失败返回false，输出值result为 0

默认值

每种类型的默认值都是0

bool型是false

引用类型默认null

方法

引用传递

参考c的指针

out参数

  • C#不允许没有初值的变量作为参数传递给方法

为了不用给一些需要通过函数获得的变量赋初值使用out

可选参数、命名参数

声明定义阶段对参数进行赋值，则该参数就为可选，不传参是使用缺省值

变长参数

当传入方法的参数数量不确定时

  • 在方法声明中的 params 关键字之后不允许任何其他参数，并且在方法声明中只允许一个 params 关键字

OOP

概念

类、对象、数据抽象、数据封装、继承、多态、动态绑定、消息传递

访问控制

5种：

  • private：私有，系统缺省
  • public：公有、无限制
  • protected：继承类可访问
  • internal：同一文件
  • protected internal
